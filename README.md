julia-examples
=============================

![Build Status](https://travis-ci.org/amilajack/julia-examples.svg?branch=master)

## Requirements
* `julia>=0.7`

## Setup
```bash
git clone https://github.com/amilajack/julia-examples
cd julia-examples
julia
# Press ']' to open package manager
(v1.0) pkg> activate .
(LP) pkg> test
```
